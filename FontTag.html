<font>

The Font tag can be used for changing the size of your text. A higher number gives larger text, and smaller numbers give smaller text.
The color of text can also be altered using <font>.

Color Hex Code In Use
red #FF0000 Example
orange #FFA500 Example
yellow #FFFF00 Example
green #008000 Example
blue #0000FF Example
purple #800080 Example
white #FFFFFF Example
black #000000 Example
